# Spacecraft (front-end)

### Pre requirements

* [Node.js](https://nodejs.org/en/download/)

### Installing

``` bash
$ git clone https://github.com/jroliveira/spacecraft.git
$ npm install
```

### How to use it

``` bash
$ npm start
```
